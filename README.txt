To use the server:
To use the server ssh into the directory it is located.
Then, run the command "make" which will compile the server.
After compiling, run the command "./server [port]" to start the server.
Recommended to make the port > 5000.

To use the client:
To use the client ssh into the directory it is located.
Then, run the command "make" which will compile the client.
After compiling, run the command"./client [server_ip_address] [port]" to start the client.
Use same port as used on server.
The client will then prompt you to respond, and list the possible commands.

Place files you want to transfer between client and/or server into the directory of
the client/server.
